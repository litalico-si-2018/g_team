# Usage
1.  `docker-compose up --build -d`
2.  `docker-machine ssh rails52 -L 8080:localhost:9000`
3.  `docker-compose run web rails db:create`

## 色々
+ `bundle install --path vendor/bundler`
+ `eval $(docker-machine env rails52)`
